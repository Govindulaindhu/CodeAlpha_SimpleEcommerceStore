# CodeAlpha_SimpleEcommerceStore
# 🛒 Simple E-Commerce Store

A basic full stack e-commerce website built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

This project was developed as part of the Full Stack Development Internship at CodeAlpha.

---

#  Features

✅ Product Listing  
✅ Product Details Page  
✅ Shopping Cart  
✅ User Registration  
✅ User Login  
✅ Order Processing  
✅ MongoDB Database Integration  
✅ Backend API Integration

# ⚙ Installation Steps

## 1️ Clone Repository

```bash
git clone YOUR_GITHUB_LINK
```

---

## 2️ Open Backend Folder

```bash
cd backend
```

---

## 3️ Install Packages

```bash
npm install
```

---

## 4️ Start MongoDB

```bash
mongod
```

---

## 5️ Run Backend Server

```bash
npx nodemon server.js
```

Server runs on:

```bash
http://localhost:5000
```

---

#  Run Frontend

Open `index.html` using Live Server in Visual Studio Code.

# 📦 API Endpoints

## Get Products

```bash
GET /products
```

## Add Product

```bash
POST /products
```

## Register User

```bash
POST /register
```

## Login User

```bash
POST /login
```

## Place Order

```bash
POST /orders
```
